# 语法

## 字符串切片

```Python
str = "0123456789"

# '234'
str[2:5]

# '23456789'
str[2:]

# '01234'
str[0:5]
str[:5]

# '0123456789'
str[0:]
str[:]

# '02468'
str[::2]

# '13579'
str[1::2]

# '9'
str[-1]

# '2345678'
str[2:-1]

# '9876543210'
str[-1::-1]
str[::-1]
```
