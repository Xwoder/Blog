# Python 重复运算符

[TOC]

# 定义

```Python
可合并的数据类型 + 可合并的数据类型
```

 可重复的数据类型通常包含

* 字符串
* 列表
* 元组

## 例

```Python
# [1, 2, 3, 4, 5, 6]
[1, 2, 3] + [4, 5, 6]                                                                       

# (False, 500, 'Time Out')
(False, 500) + ('Time Out', )                                                               

# 'Hello World'
'Hello' + ' ' + 'World'                                                                     
```
