## 布尔类型 / `Bool`

布尔类型的数据的有2种值

* True
* False

并提供了3种运算

* and
* or
* not

## 空值 / `None`

空值是 `Python` 里一个特殊的值，用 `None` 表示。

它的类型是 `None Type`，并且也是这个类型的唯一实例。

## 字符串类型 / `str`

### 定义

#### 双引号定义

```Python
"Hello World"
```

#### 单引号定义

```Python
'Hello World'
```

#### 三引号定义

```Python
"""Hello World
My name is Python"""
```

### 转换

#### `int` to `str`

```Python
i = str(10)
```

#### `str` to `list`

```Python
s = "Hello World"
l = list(s)
```
