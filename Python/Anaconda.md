# Anaconda

[TOC]

## 环境

### 启动

```shell
$ conda activate
```

```shell
$ conda activate 环境名称
```

### 停止

```shell
$ conda deactivate
```

### 列示

```shell
$ conda env list
```

## 升级

```shell
$ conda update anaconda
```

```shell
$ conda update --all
```

## 库

列出所有已经链接至 `conda` 环境的库

```bash
$ conda list
```
