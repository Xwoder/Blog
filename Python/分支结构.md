# 分支结构

[TOC]

`Python` 中的分支结构使用 `if` 语句来实现

## 单分支

```python
if <条件>:
    # code...
```

## 二分支

```python
if <条件>:
    # code...
else:
    # code...
```

### 紧凑形式

```python
<表达式1> if <条件> else <表达式2>
```

例

```python
age = 10

# You are not an adult
print("You are " + ("an adult" if age >= 18 else "not an adult"))
```

## 多分支

```python
if <条件1>:
    # code...
elif <条件2>:
    # code...
```

```python
if <条件1>:
    # code...
elif <条件2>:
    # code...
else:
    # code...
```
