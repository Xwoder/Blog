## 标签选择器

### 语法

```CSS
标签名 {

}
```

### 示例

```css
header {
  background-color: red;
  width: 100%;
  height: 100px;
}

main {
  background-color: blue;
  width: 100%;
  height: 200px;
}

footer {
  background-color: green;
  width: 100%;
  height: 80px;
}
```
