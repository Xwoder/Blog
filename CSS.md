# CSS

## 选择器

## 属性

### width

宽度

```css
width: 100px;
```

### height

高度

```css
height: 100px;
```

### background

背景颜色或图片

```css
background: red;
background: url(image/logo.png);
```

### border

```css
border: 1px solid black;
```

### border-top

```css
border-top: 10px solid red;
```

### border-left

```css
border-left: 10px solid pink;
```

### border-right

```css
border-bottom: 10px solid blue;
```

### border-bottom

```css
border-right: 10px solid green;
```

### font-family

```css
font-family: 'Consolas';
```
