# 决策树

## 信息熵（information entropy）

$$
\operatorname{Ent}(D)=-\sum_{k=1}^{|\mathcal{Y}|} p_{k} \log _{2} p_{k}
$$

含义：

|     符号      | 含义                  | 例   |
| :-----------: | --------------------- | ---- |
| $\mathcal{Y}$ | 样本类别数            |      |
|      $k$      | 第 $k$ 类样本         |      |
|     $p_k$     | 第 $k$ 类样本所占比例 |      |
