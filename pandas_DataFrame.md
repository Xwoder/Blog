## 方法

### dropna

删除空值

```python
df.dropna(how='all')
```

#### 参数

| 参数名 | 含义     |
| :----: | -------- |
| `how`  | 删除方式 |
| `axis` | 轴       |

### fillna

填充空值

```python
df.fillna(df.mean())	
```

#### 参数

|  参数名  | 含义                                         | 可选值           |
| :------: | -------------------------------------------- | ---------------- |
| `method` | 填充方式；最上方非空缺失值或最下方非空缺失值 | `ffile`、`dfill` |

### describe

数据主要特征描述，如：最大值，最小值，均值，上四分位数，下四分位数。
