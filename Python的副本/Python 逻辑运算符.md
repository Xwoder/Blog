# Python 逻辑运算符

运算符 | 描述 | 实例
:---: | :---: | ---
`and` | 与 | `(10 > 2) and (10 < 5)`
`or` | 或 | `(10 > 2) or (10 > 5)`
`not` | 非 | `not (10 > 2)`

## and

```python
age = 50

# True
age >= 0 and age <= 120
```

## or

```python
score_c = 90
score_java = 50

# True
score_c >= 60 or score_java >= 60
```

## not

```python
is_employee = True

# False
not is_employee
```
